# SaaS-O-Matic

Herramienta interna para simular, optimizar y presupuestar suscripciones SaaS multi-divisa. Backend NestJS + Prisma + SQLite · Frontend React + Vite + TypeScript.

> README provisional — se completará al cierre del proyecto (ver `README.template.md`).

## Arranque rápido

Requisitos: **Node ≥ 20** y **pnpm** (`npm i -g pnpm`).

```bash
# Terminal 1 — API (http://localhost:3000, Swagger en /docs)
cd backend && pnpm install && pnpm start:dev

# Terminal 2 — Dashboard (http://localhost:5173)
cd frontend && pnpm install && pnpm dev
```
